Para finalizar

Las cosas esenciales que debe saber hacer con las capas es:

  • Ocultarlas o apagarlas.
  • Cambiarles el nombre.
  • Cambiar el tamaño de las miniaturas.
  • Hacer aparecer y desaparecer las miniaturas.
  • Ponerles color a las capas.
  • Bloquear de alguna manera la capa para que no pueda modificarla.
